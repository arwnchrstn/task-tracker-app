# Task Tracker using MERN Stack

### Features:

- Login
- Signup
- Forgot password
- Tasks dashboard
- Profile page
- Add, update, delete categories
- Add, update, delete tasks
- Mailing functionality (For sending verification and confirmation via email)
- Delete account
- JWT authentication
- Installable (PWA)
